# 📝 Prompts para Geração de Imagens de UI - LaCuisine (Culinary Zen)

Este documento contém os prompts otimizados para gerar as telas do sistema LaCuisine seguindo a estética "Culinary Zen": minimalista, premium e focada no chef (Mobile-First).

**Paleta de Cores:**
- **Creme (Bege Claro):** `#F4EFE7`
- **Grafite (Preto Elegante):** `#111111`
- **Ouro (Dourado Suave):** `#C9A46A`

---

## Tela 1: Login / Autenticação (Mobile-First)
**Conceito:** Minimalismo absoluto, foco em conversão e clareza. Sem fotografias de fundo.
**Prompt:**
```text
Mobile app UI design of a luxury culinary SaaS login screen. Minimalist "Culinary Zen" style. Solid pale cream background (#F4EFE7). Elegant graphite black (#111111) serif typography for the logo "LaCuisine" at the top. Clean data entry fields with thin graphite borders. Primary "Sign In" button in soft rich gold (#C9A46A). No background images, lots of white space, sophisticated, high-end restaurant management feel, premium aesthetic, 8k --ar 9:16
```

## Tela 2: O Agente / Dashboard (O Chat Culinário)
**Conceito:** O dashboard é a própria interface de chat. IA-First.
**Prompt:**
```text
Mobile app UI design of an AI-driven culinary assistant chat interface. Minimalist and clean. Background in pale cream (#F4EFE7). Chat text using elegant sans-serif (Montserrat) in graphite black. No heavy message bubbles, just professional editorial-style typography. At the bottom, a sleek input bar with a golden star icon for "AI Generate". A floating hamburger menu icon at the top left. High-end, Zen, professional chef's tool, 8k --ar 9:16
```

## Tela 3: Biblioteca de Receitas (Digital Recipe Book)
**Conceito:** Grid de duas colunas, fotos gastronômicas impecáveis.
**Prompt:**
```text
Mobile app UI design of a recipe library for a premium culinary app. Two-column grid layout. Pale cream background. Large vertical recipe cards with mouth-watering gourmet photography. Below each image, elegant graphite title in Playfair Display. A small, subtle golden arrow icon at the bottom right of each card for expanding financial details. Professional, magazine-style editorial, clean and uncluttered, 8k --ar 9:16
```

## Tela 4: Card Expandido (Smart View / Financeiro)
**Conceito:** O card aberto mostrando custos e rendimento.
**Prompt:**
```text
Mobile UI component design of an expanded recipe card. Minimalist details showing "Cost per Serving" and "Yield" in soft gold and graphite typography. Thin golden lines separating data points. Pale cream background. Professional culinary management data visualization, elegant, legible, premium SaaS vibe, 8k --ar 9:16
```

## Tela 5: Ficha Técnica (Gestão de Produção)
**Conceito:** Tabela técnica refinada, otimizada para toque.
**Prompt:**
```text
Mobile app UI design of a professional culinary technical sheet. Clean table layout on pale cream background. Table headers in graphite black. Extremely thin golden horizontal separators. Monospaced numbers for weights and costs. "Edit" icons in soft gold. Editorial precision, Michelin-star restaurant back-office feel, high contrast, elegant, 8k --ar 9:16
```

---

### Dicas de Uso
- **Mobile-First:** Todos os prompts usam `--ar 9:16` para garantir o formato de smartphone.
- **Tipografia:** Playfair Display (Serif) para títulos e Montserrat (Sans) para interface.
- **Aesthetic:** Sempre use os termos "Culinary Zen", "Minimalist" e "Premium" para manter a consistência.
